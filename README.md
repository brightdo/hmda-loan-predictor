# HMDA Mortgage Loan Approval Prediction (Mississippi)

ML pipeline + interactive Tableau dashboard + Streamlit app predicting mortgage 
loan approval outcomes using 2017 HMDA data for Mississippi.

- **Live app:** https://hmda-loan-predictor.streamlit.app/
- **Tableau dashboard:** https://public.tableau.com/app/profile/do.hyeong.kim/viz/Book1_17460654931590/viz?publish=yes
- **Model:** Random Forest (ROC-AUC 0.9155), trained on 20,993 records after 
  cleaning/feature engineering/PCA (23 components)
- **Stack:** Python, scikit-learn, pandas, Streamlit, Tableau

See `DATA230_HMDA_Technical_Report.pdf` for the full technical writeup.
